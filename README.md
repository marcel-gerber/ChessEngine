## Build from source

Configure CMake.

```bash
cmake -S . -B build
```

Build executable.

```bash
cmake --build build --target ChessEngine
```

## Running Tests

Configure CMake.

```bash
cmake -S . -B build
```

Build test-executable.

```bash
cmake --build build --target ChessEngineTests
```

Run tests.

```bash
cd build/test
./ChessEngineTests
```
